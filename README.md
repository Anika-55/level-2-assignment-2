## Project Overview

Car Rental API is a backend application for managing car rentals. It provides RESTful APIs to handle authentication, vehicle management, booking, and user management. The API is built with scalability and serverless deployment in mind.

---

## Features

- User authentication (Sign Up / Login)
- Role-based access control (Admin / Customer)
- Manage cars and vehicles
- Create, update, and track bookings
- PostgreSQL database integration
- RESTful API design
- Serverless-ready deployment (Vercel)

---

## Technology Stack

- **Backend:** Node.js, Express.js, TypeScript
- **Database:** PostgreSQL
- **ORM / Query:** `pg` (node-postgres)
- **Environment Management:** dotenv
- **Deployment:** Vercel 
- **Version Control:** Git
